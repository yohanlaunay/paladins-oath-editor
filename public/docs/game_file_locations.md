File Locations for the game

=== '''Save Files''' ===
If you want to play around with the Save Files (back them up yourself, edit them etc.), they are stored in compact JSON format at a different location based on the system that you use.

====== Windows ======
<blockquote>%USERPROFILE%/AppData/LocalLow/Fire Biscuit Games/Paladin Oath/saves</blockquote>

====== Mac ======
<blockquote>~/Library/Application Support/com.FireBiscuitGames.PaladinOath/saves</blockquote>

====== Linux ======
<blockquote>~/.config/unity3d/Fire Biscuit Games/Paladin Oath/saves</blockquote>
